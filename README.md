# INC 5000 Business Analytics Project

This project focuses on analyzing the growth and revenue patterns of top U.S. companies listed in the INC 5000 dataset. The goal is to understand business performance trends using data cleaning, exploratory data analysis (EDA), and visualizations.

## Project Objectives
- Study revenue and growth patterns of selected companies.
- Identify factors that influence business growth.
- Visualize key insights to observe trends and patterns.
- Prepare a clear analytical report based on findings.

## Dataset
The dataset used in this project contains various attributes related to the INC 5000 companies, such as:
- Company name
- Industry
- Revenue
- Growth rate
- Employee count
- Location
- Year of ranking

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data loading and cleaning  
2. Handling missing values  
3. Exploratory data analysis  
4. Visualizations  
5. Insights and conclusion  

## Visualizations Included
- Revenue distributions  
- Growth rate comparisons  
- Industry-wise analysis  
- Correlation heatmaps  
- Top performing companies
